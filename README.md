## Get Started
Clone this repository to get started!
```sh
git clone https://github.com/nilaysarma/IITG-Python-Projects.git
```

## Run a Python file
To run a Python file, use this command

On Windows
```
python FILE_NAME.py
```
On Linux & MacOS
```
python3 FILE_NAME.py
```
